
# 🏦 Bank Loan Risk Analysis & Business Requirements Documentation (BRD)

## 📌 Project Overview

The **Bank Loan Risk Analysis & Business Requirements Documentation (BRD)** project is an end-to-end Data Analytics and Business Analysis initiative focused on understanding loan approval patterns and translating analytical findings into business requirements for an automated loan underwriting system.

Using historical loan application data, the project leverages **MySQL** for data analysis, **Microsoft Excel** for dashboarding and statistical analysis, and **Business Analysis techniques** to create a comprehensive Business Requirements Document (BRD). The objective is to identify key risk factors affecting loan approvals and define business rules that can support future automation of loan decision-making processes.

---

## 🎯 Project Objectives

* Analyze historical loan application data.
* Identify factors influencing loan approvals and rejections.
* Perform risk assessment using SQL and Excel.
* Create interactive dashboards for business insights.
* Develop a Business Requirements Document (BRD) for an automated loan underwriting process.
* Translate data findings into actionable business requirements.
